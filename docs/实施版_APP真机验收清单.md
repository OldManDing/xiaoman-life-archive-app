# 实施版：App 真机验收清单

更新时间：2026-05-13

## 验收前提

- API、App、Admin 均部署到 HTTPS 域名。
- API 健康检查中 `providers.storage`、`providers.ai`、`providers.map` 不为 `mock`。
- 使用真实账号密码登录；邀请码只用于注册。
- 至少准备一台 Android 和一台 iOS 设备。

## Android

- [ ] 账号密码登录成功，协议未勾选时不能登录。
- [ ] 注册页填写家庭邀请码后可创建账号，登录页不再要求邀请码。
- [ ] 发布页点击“添加照片/视频”可唤起相机或媒体选择器。
- [ ] 发布页点击“录制语音”可唤起录音或音频选择器。
- [ ] 拒绝相机、麦克风、定位权限后，页面有中文提示且可继续手动操作。
- [ ] 上传 JPG/PNG/WebP/HEIC 照片后，详情页、时间轴、首页最近更新均能预览。
- [ ] 上传 MP4/WebM/MOV 视频后，详情页和时间轴可播放。
- [ ] 上传 M4A/MP3/WAV/WebM/OGG 语音后，详情页和时间轴可播放。
- [ ] 输入地点关键字能返回真实 POI 候选，选择后保存到记录详情。
- [ ] 文字模式下上传区隐藏，发布必填校验中文提示正确。
- [ ] 返回键、底部导航、我的页子页面在小屏设备无遮挡。

## iOS

- [ ] Safari 或 WebView 中登录、注册、协议勾选流程正常。
- [ ] 发布页点击“添加照片/视频”可唤起相机或相册。
- [ ] 发布页点击“录制语音”可唤起系统支持的录音/音频选择能力；如系统只允许文件选择，需要记录设备和浏览器版本。
- [ ] 拒绝相机、麦克风、定位权限后，页面有中文提示且不会卡死。
- [ ] HEIC 照片上传后可在详情页、时间轴、首页回显。
- [ ] MOV 视频上传后可在详情页和时间轴播放。
- [ ] M4A 语音上传后可在详情页和时间轴播放。
- [ ] 地点 POI 搜索、定位当前、手动地点输入均能保存。
- [ ] 安全区、键盘弹出、底部导航和发布按钮无遮挡。

## 后台联动

- [ ] 后台媒体库能查到真机上传的图片、视频、语音。
- [ ] 后台媒体详情能查看归属孩子、上传者、关联记录和状态。
- [ ] 后台审核媒体、下架记录、AI 重试/取消均写入审计日志。
- [ ] App 端能看到后台审核或下架后的状态变化。

## 不通过处理

- 权限无法唤起：记录设备型号、系统版本、浏览器/WebView 版本、失败截图。
- 上传失败：记录媒体格式、大小、API 响应、对象存储日志。
- POI 无结果：记录关键字、城市、地图 provider 响应和 API 日志。
- 播放失败：记录 MIME、浏览器控制台、对象访问 URL 响应头。
