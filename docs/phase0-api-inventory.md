# Phase 0 API 盘点

## 范围

本文件记录当前 API 基线，作为后续继续开发和回归验证入口。Prisma schema 真源为 `apps/api/prisma/schema.prisma`。

## 已具备模块

- 健康检查：`GET /api/v1/health`
- 用户认证：账号密码登录、邀请码注册、refresh cookie、退出登录
- 用户资料：查看和修改当前用户昵称
- 孩子档案：创建、列表、详情、编辑
- 成长记录：创建、列表、详情、编辑、删除
- 媒体上传：创建上传凭证、确认上传、记录关联媒体
- AI 任务：创建记录摘要任务、查询任务详情
- 家庭协作：成员列表、创建邀请码、注册时消费邀请码、修改成员角色
- 管理后台：管理员登录、用户/孩子/记录/媒体/AI 任务/审计日志列表、用户启停

## 已修正的关键契约

- 创建/编辑记录 payload 支持 `media_nos`，前端上传后会随记录保存提交。
- 禁用用户无法登录，未过期用户 token 也无法继续访问业务接口。
- 禁用管理员无法登录，未过期管理员 token 也无法继续访问后台接口。
- `.env.example`、Docker 和 README 的 MySQL 本地端口统一为 `3307`。
