# 权限与角色说明

## 1. 角色定义

### 角色一：家庭管理员

权限：
- 管理孩子资料
- 邀请 / 移除家庭成员
- 创建 / 编辑 / 删除成长记录
- 查看全部家庭范围内可见记录
- 管理部分设置

### 角色二：家庭成员（可编辑）

权限：
- 查看成长记录
- 创建成长记录
- 编辑自己有权限的记录
- 上传媒体

### 角色三：家庭成员（只读）

权限：
- 查看相关记录
- 不可创建、编辑、删除

## 2. 权限矩阵

| 功能 | 管理员 | 可编辑成员 | 只读成员 |
|---|---|---|---|
| 查看首页 | ✅ | ✅ | ✅ |
| 查看时间轴 | ✅ | ✅ | ✅ |
| 新增记录 | ✅ | ✅ | ❌ |
| 编辑记录 | ✅ | ✅（受限） | ❌ |
| 删除记录 | ✅ | ❌ / 受限 | ❌ |
| 编辑孩子资料 | ✅ | ❌ | ❌ |
| 邀请成员 | ✅ | ❌ | ❌ |
| 修改设置 | ✅ | ✅（个人设置） | ✅（个人设置） |
