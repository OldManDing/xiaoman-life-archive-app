# 文案规范

## 1. 文案风格要求

- 温和、清晰、简短
- 不生硬、不像后台系统
- 不要出现技术化提示语
- 对家长用户友好

## 2. 成功文案示例

- 保存成功
- 记录已发布
- 宝宝资料已更新
- 邀请已发送

## 3. 错误文案示例

- 网络开小差了，请稍后重试
- 提交失败，请检查后重试
- 验证码错误，请重新输入
- 图片上传失败，请重试

## 4. 空状态文案示例

### 首页空状态
- 还没有成长记录
- 从第一条记录开始，留住每一次成长瞬间
- 按钮：立即记录

### 时间轴空状态
- 暂无时间轴内容
- 去创建一条成长记录吧
- 按钮：新增记录

### 家庭成员空状态
- 还没有家庭成员加入
- 邀请家人一起记录成长瞬间
- 按钮：邀请成员

## 5. 校验提示文案示例

- 请输入标题
- 请选择分类
- 请填写宝宝姓名
- 出生日期不能晚于今天
- 最多上传 9 张图片
