# 核心用户路径

## 1. 新用户首次使用路径

1. 启动 APP
2. 进入登录 / 注册页
3. 验证登录成功
4. 首次进入完善宝宝信息页
5. 完成孩子姓名、生日、性别、头像等基础资料
6. 进入首页
7. 首页引导用户新增第一条记录

**验收点**
- 未登录用户不可直接进入主页面
- 首次无孩子资料时必须进入引导流程
- 完善孩子资料成功后自动进入首页

## 2. 新增成长记录路径

1. 首页点击“新增记录”
2. 进入新增记录页
3. 输入标题
4. 选择分类
5. 输入内容
6. 上传图片 / 视频
7. 选择记录时间
8. 点击提交
9. 提交成功
10. 跳转到记录详情页或返回时间轴页并高亮最新记录

**验收点**
- 必填项未填写时不可提交
- 提交成功必须有明确反馈
- 提交后的记录在时间轴中可见

## 3. 时间轴查看与编辑路径

1. 进入时间轴页
2. 浏览记录列表
3. 点击一条记录进入详情
4. 点击编辑
5. 修改内容
6. 保存成功
7. 返回详情页

**验收点**
- 编辑后内容刷新可见
- 非创建者不可编辑（若权限如此定义）

## 4. 家庭成员邀请路径

1. 进入家庭页
2. 点击家庭成员管理
3. 点击邀请成员
4. 生成邀请码 / 分享链接
5. 被邀请人登录并接受邀请
6. 成为家庭成员并可查看相应内容

## 5. 我的页设置路径

1. 进入我的页
2. 打开设置
3. 修改通知开关或账号信息
4. 保存成功
