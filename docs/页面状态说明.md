# 页面状态与异常状态说明

## 1. 通用状态定义

每个关键页面原则上都要考虑以下状态：

- 初始状态
- 加载中状态
- 加载成功状态
- 空状态
- 加载失败状态
- 无权限状态
- 未登录状态
- 网络异常状态

## 2. 首页状态说明

| 状态 | 说明 | UI表现 | 用户动作 |
|---|---|---|---|
| Loading | 首页首次请求数据中 | 骨架屏 / 卡片占位 | 等待 |
| Success | 正常获取数据 | 显示概览、快捷入口、最近记录 | 正常操作 |
| Empty | 当前孩子暂无任何记录 | 展示空状态与“立即记录”按钮 | 去新增记录 |
| Error | 数据拉取失败 | 错误提示 + 重试按钮 | 点击重试 |
| Unlogin | 未登录 | 跳登录页 | 登录 |

## 3. 时间轴状态说明

| 状态 | 说明 | UI表现 | 用户动作 |
|---|---|---|---|
| Loading | 列表首次加载 | 列表骨架 | 等待 |
| Empty | 无成长记录 | 插画 + 文案 + 新增按钮 | 去新增 |
| LoadMore | 加载更多中 | 底部 loading | 等待 |
| NoMore | 已无更多数据 | 底部提示“已经到底了” | 无 |
| Error | 列表失败 | 错误态 + 重试 | 重试 |

## 4. 新增 / 编辑记录页状态说明

| 状态 | 说明 | UI表现 |
|---|---|---|
| Initial | 初始编辑态 | 空表单 / 默认值 |
| Uploading | 媒体上传中 | 缩略图 loading、进度显示 |
| DraftSaved | 草稿保存成功 | Toast 提示 |
| SubmitLoading | 正式提交中 | 提交按钮 loading 且禁用 |
| SubmitSuccess | 提交成功 | Toast + 跳转 |
| ValidationError | 表单校验失败 | 字段红字提示 / Toast |
| SubmitError | 提交失败 | Toast + 保留当前输入 |

## 5. 家庭页状态说明

- 无家庭成员时：显示引导添加成员
- 邀请失败时：显示失败原因与重试
- 无权限修改时：禁用相关按钮并说明原因
