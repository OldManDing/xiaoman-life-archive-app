# 验收标准

更新时间：2026-05-28

## 1. 版本验收范围

本期必须完成页面：

1. 登录 / 注册页
2. 完善宝宝信息页
3. 首页
4. 时间轴页
5. 新增记录页
6. 记录详情页
7. 编辑记录页
8. 家庭页
9. 家庭成员管理页
10. 我的页
11. 月报 / 纪念册页
12. 档案导出与交付申请页
13. 帮助反馈页
14. 关于、隐私、协议、账号安全和设置页
15. 管理后台总览、账号、家庭、孩子、记录、媒体、邀请码、AI 任务、内容风险、客服反馈、档案交付、系统运维、系统配置和审计页面

## 2. 本期必须打通的核心流程

1. 登录 / 注册 -> 完善宝宝信息 -> 首页
2. 首页 -> 新增记录 -> 提交成功 -> 记录可在时间轴查看
3. 时间轴 -> 详情 -> 编辑 -> 保存成功
4. 家庭页 -> 家庭成员管理 -> 邀请成员
5. 我的页 -> 月报 / 纪念册 -> 档案导出 -> 正式打包或成年移交申请
6. 我的页 -> 帮助反馈 -> 后台客服反馈队列可查询并推进状态
7. 后台 -> 账号管理 -> 调整套餐权益并写入审计
8. 后台 -> 内容风险 / 系统运维 / 系统配置 -> 能定位风险、查看发布门禁和维护备份告警配置

## 3. UI 验收标准

- 页面整体布局与设计稿基本一致
- 字体层级、留白、圆角、阴影、边框风格统一
- 不得出现明显默认模板味过重的问题
- 底部导航、页面标题区、表单、卡片风格统一
- 空状态、加载状态、错误状态具备

## 4. 功能验收标准

- 主要页面均可进入
- 关键按钮均可点击并有正确行为
- 表单校验符合规则
- 提交成功后有明确反馈
- 列表能正常展示与刷新
- 详情页与编辑页数据一致
- 用户端高德 POI 失败时必须保留手动地点发布兜底，并给出中文反馈

## 5. 工程验收标准

- 项目可运行
- 页面无严重报错
- 路由可正常访问
- 代码结构清晰
- 组件拆分合理
- 核心用户端和后台接口已接入真实 API、审计和权限校验
- 发布前门禁覆盖生产模板、安全基线、依赖审计、回归测试、构建、浏览器 E2E 和 live readiness 报告

## 6. 可延期项

本期允许延期但不得伪装完成：

- 复杂消息通知中心
- 复杂客服 SLA、分派和通知中心
- 高级筛选组合
- 多端深度同步策略
- 智能推荐类功能

已解除的延期项：

- 高德真实 POI Key 已接入生产并通过 `verify:live-readiness`，P1-03 不再按延期项处理；如后续更换 key、provider 或服务器出口策略，必须重新复验。
