# 验收标准

## 1. 版本验收范围

本期必须完成页面：

1. 登录 / 注册页
2. 完善宝宝信息页
3. 首页
4. 时间轴页
5. 新增记录页
6. 记录详情页
7. 编辑记录页
8. 家庭页
9. 家庭成员管理页
10. 我的页
11. 关于 / 设置未开放说明页

## 2. 本期必须打通的核心流程

1. 登录 / 注册 -> 完善宝宝信息 -> 首页
2. 首页 -> 新增记录 -> 提交成功 -> 记录可在时间轴查看
3. 时间轴 -> 详情 -> 编辑 -> 保存成功
4. 家庭页 -> 家庭成员管理 -> 邀请成员
5. 我的页 -> 未交付能力明确降级为规划中/待接入

## 3. UI 验收标准

- 页面整体布局与设计稿基本一致
- 字体层级、留白、圆角、阴影、边框风格统一
- 不得出现明显默认模板味过重的问题
- 底部导航、页面标题区、表单、卡片风格统一
- 空状态、加载状态、错误状态具备

## 4. 功能验收标准

- 主要页面均可进入
- 关键按钮均可点击并有正确行为
- 表单校验符合规则
- 提交成功后有明确反馈
- 列表能正常展示与刷新
- 详情页与编辑页数据一致

## 5. 工程验收标准

- 项目可运行
- 页面无严重报错
- 路由可正常访问
- 代码结构清晰
- 组件拆分合理
- 具备后续接入真实接口的能力

## 6. 可延期项

本期允许延期或先 mock：

- 复杂消息通知中心
- 用户反馈工单系统
- 高级筛选组合
- 多端深度同步策略
- 智能推荐类功能
