# 非功能需求

## 1. 性能要求

- 首页首屏应尽可能快速加载
- 列表滚动流畅
- 图片缩略图加载需优化
- 提交按钮需具备防重复提交能力

## 2. 兼容性要求

- 适配主流 iOS / Android 手机尺寸
- 常见全面屏安全区适配
- 字体与布局在主流设备下不应错位

## 3. 稳定性要求

- 异步请求需统一处理错误
- 页面异常不应直接白屏
- 网络失败需允许用户重试

## 4. 安全与隐私要求

- 登录态需安全存储
- 用户协议与隐私政策需可查看
- 孩子隐私数据不得在前端无保护暴露
- 媒体访问需具备合理权限控制

## 5. 可维护性要求

- 路由清晰
- 页面拆分合理
- 通用组件抽取到位
- 样式体系统一
- 类型定义明确
