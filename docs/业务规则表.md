# 业务规则表

## 1. 通用业务规则

| 编号 | 规则名称 | 规则说明 |
|---|---|---|
| R001 | 登录前置 | 未登录用户不可访问首页、时间轴、家庭、我的等主业务页 |
| R002 | 孩子前置 | 若账号下无孩子资料，则登录后优先进入“完善宝宝信息”流程 |
| R003 | 当前孩子上下文 | 首页、时间轴、记录创建默认绑定当前孩子 |
| R004 | 多孩子切换 | 若账号下存在多个孩子，需支持切换当前孩子上下文 |
| R005 | 软删除优先 | 记录删除默认采用软删除，避免用户误删无法恢复 |
| R006 | 时间记录合法性 | 记录时间允许选择过去时间，默认不允许未来时间 |
| R007 | 重复提交防抖 | 提交按钮点击后进入 loading，接口返回前禁止重复提交 |

## 2. 登录注册规则

| 编号 | 规则名称 | 规则说明 |
|---|---|---|
| A001 | 协议勾选 | 未勾选用户协议与隐私政策时，登录按钮不可用或点击后提示 |
| A002 | 账号密码 | 用户端使用账号密码登录，密码不得明文保存 |
| A003 | 邀请码注册 | 注册账号时需使用有效家庭邀请码 |
| A004 | 首次登录识别 | 登录或注册成功后，如无可访问孩子资料则进入完善宝宝信息流程 |

## 3. 孩子资料规则

| 编号 | 规则名称 | 规则说明 |
|---|---|---|
| C001 | 孩子姓名必填 | 姓名必填，长度 1-20 字 |
| C002 | 出生日期必填 | 出生日期必填，不能晚于当前日期 |
| C003 | 头像选填 | 可不上传头像，默认使用系统占位头像 |
| C004 | 性别选填 | 支持男 / 女 / 未设置 |
| C005 | 多孩支持 | 若支持多孩，每个孩子数据隔离 |

## 4. 成长记录规则

| 编号 | 规则名称 | 规则说明 |
|---|---|---|
| G001 | 标题必填 | 记录标题必填，长度 1-50 字 |
| G002 | 分类必填 | 每条记录至少选择 1 个分类 |
| G003 | 内容选填 | 记录正文可选填，长度上限 2000 字 |
| G004 | 标签选填 | 支持多标签，最多 10 个 |
| G005 | 图片上限 | 最多上传 9 张图片 |
| G006 | 视频上限 | 默认最多上传 1 个视频 |
| G007 | 媒体格式限制 | 图片支持 jpg/png/webp；视频支持 mp4/mov |
| G008 | 时间可编辑 | 记录时间可修改，但不能超过当前时间 |
| G009 | 草稿保存 | 草稿允许缺少部分必填字段，但正式提交必须校验 |
| G010 | 草稿展示范围 | 草稿不出现在正式时间轴列表中 |
| G011 | 提交成功处理 | 提交成功后提示“保存成功”，并跳详情页或返回时间轴 |
| G012 | 编辑权限 | 仅记录创建者或具备编辑权限的家庭成员可编辑 |
| G013 | 删除权限 | 仅记录创建者或具备删除权限的家庭成员可删除 |
| G014 | 删除提示 | 删除前必须二次确认 |
| G015 | 删除结果 | 删除后列表与详情状态同步更新 |

## 5. 家庭协作规则

| 编号 | 规则名称 | 规则说明 |
|---|---|---|
| F001 | 家庭至少一名管理员 | 每个家庭至少保留 1 名管理员 |
| F002 | 邀请有效期 | 邀请码默认 24 小时有效 |
| F003 | 关系字段必填 | 家庭成员加入时需设置与孩子关系 |
| F004 | 权限范围 | 家庭成员权限可配置为“仅查看”或“可编辑” |
| F005 | 移除成员 | 管理员可移除普通成员，被移除成员立即失去访问权 |

## 6. 个人设置规则

| 编号 | 规则名称 | 规则说明 |
|---|---|---|
| S001 | 昵称限制 | 昵称长度 1-20 字 |
| S002 | 头像可修改 | 用户头像允许替换 |
| S003 | 通知开关即时生效 | 修改通知开关后立即保存并生效 |
